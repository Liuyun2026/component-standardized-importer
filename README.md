# component-standardized-importer
为了解决Kicad库支持不齐全，以及嘉立创有的元件找不到是的需要单独导入，元件库标准化，元件管理困难的问题，我做了一个小工具，希望帮助大家更方便更无痛的使用Kicad画板子
功能如下：
1.可以将一个包含3D，封装，元件符号的文件夹（不管里面嵌套几层，小工具可以自己找到对应的文件）直接整合进库里，并放到相应位置
2.由于元件符号库不是文件夹而是一个.sym的文件，小工具支持自主按照库里已有的元件的语法修改格式并自主合并
3.根据提示提供货源平台网址，自动识别抓取数据手册，描述，封装等信息，以及该器件在平台的编号，方便配型
4.提供3D模型坐标修改表，如果你发现3D模型位置不对请修改调整表返回，该器件将从库更新，弥补了kicad放置多个相同器件（坐标不对）需要一个一个改的问题，一劳永逸

目前已经更新到v1.0.3（测试到v1.0.2，v1.0.3）没测试

注：v1.0.1 v1.0.2 是skill，需结合agent使用，推荐使用WorkBuddy，v1.0.3有.py脚本（未测试）
